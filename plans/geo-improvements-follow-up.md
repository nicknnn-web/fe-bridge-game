# GEO Base 改进方案后续优化建议

基于「建议.txt」详细需求，以下是可进一步改进的要点：

---

## 一、导航结构优化

### 当前问题
建议中的导航更细致，缺少以下细分页面：

```
品牌监控（当前只有一个页面，建议拆分为4个）
├── 企业品牌查询（单次 GEO 检测）- ✅ 已有（brand.html）
├── 企业品牌监控（持续监控配置）- ❌ 缺少
├── 企业管理设置（监控任务与竞争对手配置）- ❌ 缺少
└── 品牌管理设置（品牌资产库管理）- ❌ 缺少

历史记录 - ❌ 缺少整个菜单
├── 检测历史（可展开下拉）

业务主题（当前混在一起，建议拆分）
├── 热门业务主题（热点趋势风向标）- ❌ 独立页面缺少
└── AI对话问题（用户问题列表与表现）- ❌ 独立页面缺少
```

### 建议改进
1. **拆分品牌监控为4个独立页面**
2. **添加历史记录菜单**
3. **拆分业务主题为2个独立页面**

## 三、新增页面需求

### 1. 企业品牌监控（持续监控）- 全新页面
**布局**：顶部 Tab 切换

**Tab 1 - 通过品牌描述监控**：
- 品牌名称输入框
- 品牌描述文本域（最少20字限制，字数统计 0/599）
- 产品服务标签输入（至少2个标签）

**Tab 2 - 通过网站地址监控**：
- 品牌名称输入框
- 品牌官网URL输入框
- 「开始分析」蓝色大按钮 + 小字说明

### 2. 企业管理设置 - 全新页面
**双标签页设计**：
- **标签页1 - AI对话问题设置**：筛选栏 + 数据表格
- **标签页2 - 竞争对手设置**：搜索框 + 新增按钮 + 数据表格

### 3. 品牌管理设置 - 全新页面
**标准数据表格**：
- 列：序号、品牌名称、别名、任务名称、描述、产品服务（标签）、URL、状态、到期时间、操作

### 4. AI对话问题挖掘 - 全新页面
**特殊布局设计**：
- 上半部分：大搜索区（输入框 + 快捷示例标签 + 立即查询按钮）
- 下半部分：三列价值卡片（精准定位信息、打破信息孤岛、辅助决策洞察）

### 5. 热门业务主题 - 独立页面
**数据表格**：序号、主题名称、平均位置、舆论评价、曝光度、引用最高、操作

### 6. AI对话问题 - 独立页面
**数据表格**：序号、AI对话问题、热度指数（可排序）、曝光度、平均位置、舆论评价、前三名、操作

### 7. 历史记录 - 全新页面
**可展开下拉列表**：展示所有检测历史记录

## 四、交互规范完善

### 1. 数据导出功能
**建议要求**：所有图表需支持数据导出（CSV/PNG）功能入口
- 当前：❌ 缺少导出按钮
- 建议：在图表右上角添加「导出」下拉按钮

### 2. 分页器规范
**建议要求**：底部居中分页器（页码 + 上一页/下一页 + 每页条数选择）
- 当前：❌ 大部分表格没有分页
- 建议：统一添加分页组件

### 3. 表单交互细节
**步骤指示器**：
- 蓝色高亮当前步骤
- 灰色显示未完成步骤
- 已完成步骤显示蓝色勾选

**标签输入**：
- 回车生成蓝色圆角标签
- 带 X 删除按钮
- 支持重复校验提示

## 五、视觉规范对齐

### 颜色规范差异
| 元素 | 建议规范 | 当前状态 |
|------|---------|---------|
| 主色调 | #3B82F6 | ✅ 一致 |
| 标题文字 | #333333 | ⚠️ 使用 #1F2937 |
| 正文文字 | #666666 | ⚠️ 使用 #4B5563 |
| 次要文字 | #999999 | ⚠️ 使用 #9CA3AF |
| 卡片圆角 | 24px | ⚠️ 使用 12-16px |
| 表头背景 | #FAFAFA | ⚠️ 使用透明 |

### 时间选择器
- **建议**：统一放置在页面右上角，默认最近7天
- **当前**：部分页面有，但位置和样式不统一
