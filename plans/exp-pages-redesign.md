# 实验功能页面重新设计方案

## 设计目标
- 三个实验功能页面视觉风格统一
- 每个页面都包含数据可视化展示
- 整体色调保持一致（基于现有深色主题）
- 提升用户体验和信息展示效率

## 三、统一组件库

### 3.1 图表组件规范
所有图表使用 ECharts，统一配置主题：

```javascript
// 统一图表主题配置
const chartTheme = {
    color: ['#6366F1', '#10B981', '#0EA5E9', '#F59E0B', '#8B5CF6', '#EF4444'],

    textStyle: { color: '#94A3B8' },
    title: { textStyle: { color: '#F1F5F9' } },
    legend: { textStyle: { color: '#94A3B8' } },
    tooltip: {
        backgroundColor: '#1E293B',
        borderColor: 'rgba(99, 102, 241, 0.3)',
        textStyle: { color: '#F1F5F9' }
    }
};
```

## 四、实施建议

### 4.1 优先级
1. **P0** - 统一色彩系统和卡片样式
2. **P1** - 添加基础数据可视化（统计卡片、进度条）
3. **P2** - 添加高级图表（雷达图、趋势图）
4. **P3** - 添加动画和交互细节

### 4.2 技术方案
- 使用现有 `echarts-utils.js` 封装图表组件
- 在 `exp-pages.css` 中添加统一样式变量
- 每个页面独立引入需要的图表配置

### 4.3 兼容性
- 保持与现有 localStorage 数据格式兼容
- 渐进式增强，不破坏现有功能
- 响应式设计，适配不同屏幕尺寸
