# 火焰纹章：断桥之战 — 地图重设计方案

## 1. 战略背景

蓝狮王国在边境矿区布下环形防御阵地，红狼帝国从**北线**（主力）和**东线**（侧翼）两路夹击。战场被西北→东南的河流斜线分割，形成天然分战区。

## 4. 战术设计

### 地形价值

| 地形 | 坐标 | 战术意义 |
|------|------|---------|
| 蓝城要塞 | (0-1,9-10) | 蓝方环形阵地核心，DEF+4，4格连片 |
| 红城 | (10-11,0-1) | 红方北线基地，DEF+4，提供进攻起点 |
| 北森林带 | (2-3,2-3) | 红方北线进攻掩护，森林 DEF+2 |
| 东森林墙 | (8-9,5) | 蓝方东翼防线，阻挡红方东线侧翼 |
| 河流斜线 | (0,4)(10,5)(11,6) | 战场分割线，阻止直线推进，桥梁是唯一通道 |
| 中部平原 | (4-9,4-9) | 自由交战区，高机动单位（骑士）优势区域 |

### 进攻路线

**红方北线**（e1/e2）：沿地图顶部向东推进，过平原绕河流源头包抄蓝城侧翼
**红方东线**（e3/e4）：沿东侧平原南下，绕森林墙从东南方向夹击蓝城

### 防守策略

蓝方环形阵地：领主+重甲守正面（北侧），骑士游走侧翼，弓手远程压制森林通道

---

## 5. 关键设计原则

- **不对称**：地形分布刻意不规则，强迫红方两路进攻而非一字平推
- **河流分界**：斜线河流制造天然战场分割，蓝方不能直接支援东翼
- **环形要塞**：蓝城 2×2 连片，提供强防御核心
- **东翼风险**：蓝方东翼只有森林，一旦被突破缺乏纵深
